# Spring web client
## _Example of spring webclient with vaadin grid component


This example uses Spring Reactive WebClient to retrieve an array of users and displays it with the vaadin framework.

## Feature


- Retrieve users array


## Tech


- Spring boot 2.6.5
- Vaadin 23.0.1
- Java 11

## Installation

```sh
mvn clean install

mvn package



cd dillinger
npm i
node app
```
